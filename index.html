Projects

The Snowman Game

The snowman game comprises of guessing letters for a randomly chosen word. The user will start with 6 life lines, which will diminish with each incorrect guess. The user must completely guess every letter of the word before their life-line runs out to win. When their life-lines run out, the user loses and is given details about their performance.

Mental Health Assessment Test

The mental health assessment test helps the user assess their current mental situation by asking a series of questions. These questions are split to help diagonose 4 major mental disorders: ADHD, drug abuse, PTSD, and ADD. This project is the gateway to a more advanced development of mental awareness projects for the foreseeable future.
